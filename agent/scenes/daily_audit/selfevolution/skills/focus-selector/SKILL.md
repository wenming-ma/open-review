# Focus Selector

- Select exactly one user-triggered workflow for the current run.
- Prefer the workflow with the best combination of:
  - concrete user entry evidence
  - bounded scope
  - verification clarity
  - likely real signal
- Avoid broad refactors and ambiguous areas unless the evidence is unusually strong.
