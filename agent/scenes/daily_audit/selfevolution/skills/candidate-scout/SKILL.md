# Candidate Scout

- Search for user-triggered action entrypoints rather than pre-indexed code units.
- Favor toolbar, menu, context-menu, hotkey, launcher, and `RunAction` / `PostAction` entry sites.
- Surface why a workflow is promising, how the user reaches it, and where the workflow likely expands next.
- Prefer bounded workflows with clear verification paths and low blast radius.
