# Candidate Scout

- Search for user-facing or externally triggered entrypoints rather than pre-indexed code units.
- Favor HTTP routes, RPC handlers, CLI commands, scheduled jobs, queue consumers, UI actions, workflow definitions, plugin hooks, command dispatch sites, and other concrete entry sites.
- Surface why a workflow is promising, how the user reaches it, and where the workflow likely expands next.
- Prefer bounded workflows with clear verification paths and low blast radius.
